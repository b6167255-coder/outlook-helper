> 🚀 אפליקציית ווב מבוססת Flask ליצירה אוטומטית של טיוטות מייל ב-Outlook עבור מספר נמענים, כולל קובץ מצורף.

---

## 🧭 סקירה כללית

האפליקציה מאפשרת להזין פרטי מייל פעם אחת — וליצור טיוטות מוכנות ב־Outlook עבור רשימת נמענים.

במקום לכתוב את אותו מייל שוב ושוב, המערכת יוצרת טיוטות מוכנות לעריכה — חוסכת זמן ומפחיתה טעויות.

---

## ✨ תכונות עיקריות

- 📩 יצירת טיוטת Outlook נפרדת לכל נמען  
- 👥 תמיכה במספר נמענים (פסיקים / שורות / ;)  
- 📎 צירוף קובץ (PDF, DOC, DOCX, TXT וכו')  
- 🖥️ ממשק משתמש נוח וברור (תמיכה ב-RTL)  
- ⚠️ טיפול בשגיאות והצגת הודעות למשתמש  
- 🔒 שליטה מלאה — אין שליחה אוטומטית  

---

## 🛠️ טכנולוגיות

| שכבה        | טכנולוגיה            |
|------------|----------------------|
| Backend    | Python, Flask        |
| אינטגרציה  | pywin32 (COM API)    |
| Frontend   | HTML, CSS            |
| כלים נוספים | Werkzeug, tempfile   |

---

---

## ⚙️ דרישות מערכת

- מערכת הפעלה Windows  
- Microsoft Outlook מותקן  
- Python 3.8 ומעלה
